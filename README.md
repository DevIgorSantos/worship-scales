# Worship Scales

Sistema de gestão de escalas, repertório e equipes para ministérios de louvor.

## Funcionalidades

- **Gestão de Escalas**: Agendamento de cultos, notificação de equipes e confirmação de presença.
- **Repertório**: Biblioteca de músicas (Louvores e Harpa Cristã) com letras e cifras.
- **Dashboard**: Visão geral de próximos eventos, versículo do dia e estatísticas.
- **Harpa Cristã**: Integração com os 640 hinos da Harpa Cristã.

## Créditos

- **Harpa Cristã (JSON)**: Agradecimentos especiais a [Daniel Liberato](https://github.com/DanielLiberato) pela disponibilização da biblioteca [Harpa-Crista-JSON-640-Hinos-Completa](https://github.com/DanielLiberato/Harpa-Crista-JSON-640-Hinos-Completa), que possibilitou a integração dos hinos neste projeto.
